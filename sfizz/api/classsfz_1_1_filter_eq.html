Detailed Description

template<unsigned NCh>
class sfz::FilterEq< NCh >

Equalizer filter for SFZ v1 Available for mono and stereo.

(NCh=1, NCh=2)

Parameters: cutoff: it's the opcode egN_freq (Hz) bw: it's the opcode eqN_bw (octave) pksh: it's the opcode eqN_gain (dB)

Process one cycle of the filter without modulating cutoff or bandwidth.

cutoff is a frequency expressed in Hz. bw is a bandwidth expressed in octaves. pksh is a peak/shelf gain expressed in dB. in[i] and out[i] may refer to identical buffers, for in-place processing
