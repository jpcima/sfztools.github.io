◆ fp_exponent()

template<class F >
int fp_exponent ( F  x)
inline

Get the exponent part of a IEEE754 floating point number.

The number is reconstructed as (-1^sign)*(1+mantissa)*(2^exponent). See also fp_sign and fp_mantissa.

It is a faster way of computing floor(log2(abs(x))).
