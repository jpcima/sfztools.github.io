Detailed Description

template<class Type, unsigned int MaxChannels = sfz::config::numChannels, unsigned int Alignment = SIMDConfig::defaultAlignment>
class sfz::AudioBuffer< Type, MaxChannels, Alignment >

A class to handle a collection of buffers, where each buffer has the same size.

Unlike AudioSpan, this class owns its underlying buffers and they are freed when the buffer is destroyed.

Template Parameters
Typethe underlying type of the buffers
MaxChannelsthe maximum number of channels in the buffer
Alignmentthe alignment for the buffers
