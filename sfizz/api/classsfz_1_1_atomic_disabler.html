sfz::AtomicDisabler Class Reference

#include <AtomicGuard.h>

Detailed Description

Simple class to set an atomic to false and automatically set it back to true on destruction.

You call it like this assuming you need to disable e.g. a callback

void functionThatDisableAnotherFunction()
{
AtomicDisabler { disabler };
// Do stuff, the atomic will be set back to true as soon as you're back
}

Note that this is not thread-safe at all, in the sense that it is only meant to be used with 2 threads along with the AtomicGuard. One thread uses AtomicGuards, the other AtomicDisabler, and no other contending thread can share this pair of atomics.
